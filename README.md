# Ledgerflow

Ledgerflow is an AI-driven, offline-first operating system transforming the $1.4 trillion informal retail sector. By replacing manual paper records with a localized credit risk engine, Ledgerflow captures undocumented transaction data and converts messy physical ledgers into structured data pipelines—instantly connecting informal merchants to global B2B supply chains and banking infrastructure.

## Tech Stack
* **Frontend:** React, HTML, CSS, Bootstrap
* **Backend & Database:** Node.js, Express, Local Database Storage
